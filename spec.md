UI-Tool to post-process PDF files containing batches of scanned documents. It should display the pages of the input file and allow to split its pages into separate PDF files. The user should be able to choose at which page of the input file a new output file should be started and what its name and folder should be. At the end the tool should split the input file into output files according to the specification of the user.

Optionally the tool should integrate Google Drive: The user should be able to choose the Drive folder into which each output file should be uploaded. After splitting the file the tool should automatically upload the files.

Further features:

- rotation of individual pages
- insert pages out of a different PDF (not sure if/where that would fit)

- Maybe even implement the scanning process? not sure that's feasible
